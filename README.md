# 🛒 GitHub Shop - Boutique E-commerce sur GitHub

Boutique en ligne complète hébergée sur **GitHub Pages** avec **GitHub API** comme backend et **GitHub Actions** pour l'automatisation.

## ✨ Fonctionnalités

- 🛍️ Catalogue produits avec catégories, recherche, filtres
- 🛒 Panier persistant (localStorage)
- 💳 Checkout avec Stripe/PayPal
- 👤 Authentification (GitHub OAuth + email)
- 📦 Gestion des commandes
- 🎨 Panel administrateur
- 📧 Notifications email automatiques (via GitHub Actions)
- 📊 Statistiques de vente
- 🌙 Mode sombre/clair
- 📱 Design responsive (mobile-first)
- 🌍 Multi-langue (FR/EN)
